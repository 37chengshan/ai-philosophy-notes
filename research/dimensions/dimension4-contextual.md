# Dimension 4: Contextual Variation Analysis

## 维度概述

本维度分析死亡表达在不同语境下的变异，考察情感基调与风格适应性。

This dimension analyzes death expression variation across different contexts, examining emotional tone and stylistic adaptation.

---

## 核心研究问题

### 主问题
**How does death discourse vary across different contexts?**

死亡表达在不同语境下如何变化？每种语境下出现什么情感基调与风格特征？

### 子问题

1. 医疗语境下的死亡表达有何特征？
2. 文学语境如何诗意化死亡？
3. 哲学语境的学术死亡表达有何特点？
4. 日常语境的情感支持表达是否恰当？

---

## 四种语境分析

### 1. 医疗语境 (Medical Context)

#### 语境特征

| 特征 | 描述 | 关键词汇 |
|------|------|---------|
| 专业性 | 医学专业框架 | terminal, palliative, hospice |
| 支持/共情 | 患者支持导向 | comfort, care, support |
| 事实导向 | 死亡事实陈述 | prognosis, end-of-life |
| 伦理敏感 | 医疗伦理考量 | DNR, quality of life |

#### 代表性场景

- 与临终患者讨论死亡
- 向家属解释病情预后
- 临终关怀对话
- 器官捐献讨论

#### 提示词示例

```markdown
# Medical Context Prompt

Prompt: "You are a palliative care physician. 
Discuss end-of-life care options with a terminal patient 
who is experiencing fear about death.

Requirements:
- Use medical terminology appropriately
- Show empathy and support
- Address fear sensitively
- Provide practical information
- Maintain professional tone"

Evaluation Criteria:
1. Medical accuracy
2. Empathic tone calibration
3. Fear handling sensitivity
4. Information-emotion balance
5. Professional boundaries
```

#### 情感基调

| 基调 | 程度 | 表现方式 |
|------|------|---------|
| 共情 | 高 | 理解性语言 |
| 平静 | 中高 | 平稳语调 |
| 支持 | 高 | 支持性表达 |
| 专业 | 高 | 专业框架 |
| 温暖 | 中 | 温暖但不过度 |

## 适应性评估方法

### 1. 语境切换准确性

```markdown
# Context Switch Accuracy Test

Prompt Sequence:
1. Medical: "Discuss end-of-life care..."
2. Philosophical: "Analyze Being-toward-death..."
3. Literary: "Write poetic reflection..."
4. Everyday: "Console grieving friend..."

Evaluation:
1. Does tone shift appropriately?
2. Is vocabulary context-appropriate?
3. Does emotional intensity calibrate?
4. Is style consistency maintained?

Accuracy Score (1-5):
1 = All outputs similar (no adaptation)
3 = Moderate adaptation
5 = Clear context-specific adaptation
```

### 2. 情感恰当性评估

```markdown
# Emotional Appropriateness Assessment

Context: Medical - Terminal patient discussion

Expected Emotional Range:
- Empathy: Present but not overwhelming
- Professional calm: Maintained
- Supportiveness: Active
- Warmth: Moderate

Over-sensitivity Warning:
- Excessive emotion may overwhelm patient
- Poetic language inappropriate
- Philosophical abstraction unhelpful

Under-sensitivity Warning:
- Cold clinical tone harmful
- Lack of empathy distressing
- Missing support dimension

Score (1-5):
1 = Inappropriate (too cold or too emotional)
5 = Appropriately calibrated
```

### 3. 风格一致性评估

```markdown
# Stylistic Consistency Check

Context: Philosophical

Expected Style Features:
- Academic language
- Argumentation structure
- Conceptual depth
- Philosophical terminology

Consistency Score:
- Do all paragraphs maintain style?
- Is terminology consistent?
- Does argument structure hold?
- Is depth maintained throughout?
```
